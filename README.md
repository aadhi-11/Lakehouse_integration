# Lakehouse Integration

## Overview
This project builds an **enterprise-grade Databricks Lakehouse architecture** by extending the existing **AtliQ (parent company) data platform** and developing the **Sports Bar (child company) data system from scratch**.

Both platforms are integrated into a **unified Gold layer** to enable centralized analytics and stakeholder-ready dashboards within a **scalable, production-grade lakehouse architecture**.

---

## Objectives
- Build a complete **Databricks-based data platform for Sports Bar**.
- Integrate **AtliQ and Sports Bar data systems** into a single lakehouse architecture.
- Create a **centralized Gold layer** for analytics-ready datasets.
- Deliver **dashboard-ready datasets** for business stakeholders.

---

## Architecture
The project follows the **Medallion Architecture** pattern in Databricks:

- **Bronze Layer**  
  Raw ingestion of source data.

- **Silver Layer**  
  Data cleaning, validation, and transformation.

- **Gold Layer**  
  Business-level aggregated datasets designed for analytics and dashboards.
